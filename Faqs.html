What is a Fork?

In Software Programming a Fork is when someone take a copy of source code from one software package, in this case ABP 2.0, and start independent development on it, creating a separate piece of software.
A very similar Example of a fork like Adblock Edge is GNUicecat which re branded and added additional privacy features to Mozilla Firefox.
