  • Preferences

    The following table lists and explains all preferences that are used by Adblock edge. These preferences can be viewed and changed via about:config.

    Preference Default value Comments
    Visible options
    extensions.adblockedge.enabled true Value of the "Enable Adblock edge" option
    extensions.adblockedge.fastcollapse false Reverse of the "Hide placeholders of blocked elements" option
    extensions.adblockedge.frameobjects true Value of the "Show tabs on Flash and Java" option
    extensions.adblockedge.savestats true Value of the "Count filter hits" option
    extensions.adblockedge.showinstatusbar false Value of the "Show in status bar" option
    Hidden options
    extensions.adblockedge.clearStatsOnHistoryPurge true If true, clearing browsing history will also clear hit statistics for Adblock edge filters. Note that the hit statistics often allow deducing which sites the user visited, when and how often.
    extensions.adblockedge.composer_default 2 Determines which suggestion should be selected in the filter composer by default:

    0 — custom pattern
    1 — exact address only
    2 — entire directory
    3 — entire site
    extensions.adblockedge.data_directory adblockedge The directory where Adblock edge should store its data. This can be either an absolute path, or it can a directory location relative to the user's profile. If patternsfile preference is set it will override the location for patterns.ini file while other files stay in this directory.
    extensions.adblockedge.defaultstatusbaraction 0 Defines the action that should be executed when the status bar icon is clicked. For list of possible values see defaulttoolbaraction.
    extensions.adblockedge.defaulttoolbaraction 0 Defines the action that should be executed when the toolbar icon is clicked. Possible values are:

    0 — show menu
    1 — open/close blockable items
    2 — show Preferences dialog
    3 — enable/disable Adblock edge
    anything else — do nothing
    extensions.adblockedge.enable_key empty Defines the keyboard shortcut to be used to enable/disable Adblock edge. For description of possible values see sidebar_key.

    Note: This setting will only apply after the browser is restarted.
    extensions.adblockedge.filters_key Accel Shift E, Accel Shift F, Accel Shift O Defines the keyboard shortcut to be used to open Filter Preferences dialog. For description of possible values see sidebar_key.

    Note: This setting will only apply after the browser is restarted.
    extensions.adblockedge.flash_scrolltoitem true Defines whether "Flash item's borders" in the list of blockable items should try to scroll the item into view.
    extensions.adblockedge.patternsbackupinterval 24 Minimal interval between two backups of patterns.ini (in hours)
    extensions.adblockedge.patternsbackups 5 Number of backups for patterns.ini that should be kept. If this is 0, backups will be disabled.
    extensions.adblockedge.patternsfile empty Name of the file that should be used to store the list of filters. This can be either an absolute path, or it can be file's location relative to the user's profile.
    extensions.adblockedge.previewimages true Defines whether the tooltip in the list of blockable items should show the image that the item represents.
    extensions.adblockedge.sidebar_key Accel Shift V, Accel Shift U Defines the keyboard shortcut to be used to open blockable items. This preference can be empty or it can be of the form modifier1 modifier2 key. Recognized modifiers are: Shift, Accel (default accelerator key, e.g. Ctrl on Windows, Command on Mac OS X), Ctrl/Control, Alt, Meta. Key can be either a letter or a special key like F7 or RIGHT (see nsIDOMKeyEvent interface for a list of special keys).

    Note: This setting will only apply after the browser is restarted.
    extensions.adblockedge.subscriptions_autoupdate true Defines whether filters in filter subscriptions will be updated automatically.

    Note: Disabling updates is not recommended.
    extensions.adblockedge.synchronizationinterval 24 Minimal interval between two subsequent downloads of a subscription (in hours)
    Remembered actions
    extensions.adblockedge.blockableItemsSize {width: 200, height: 200} Remembered size of the Blockable Items list (width setting is only relevant when displayed as a sidebar).
    extensions.adblockedge.currentVersion 0.0 Stores the last used Adblock edge version, this will trigger actions that need to be done on upgrade or first install
    extensions.adblockedge.detachsidebar false Stores state of the list of blockable items — true means detached
    extensions.adblockedge.hideContributeButton false This is set to true if the user chose to hide the Contribute button.
    Internal preferences (don't change unless you really know what you are doing)
    extensions.adblockedge.documentation_link
    		Address on adblockedge.org to be opened for documentation links. It will redirect to the actual documentation location depending on the language used.
    extensions.adblockedge.subscriptions_listurl
    		Address to download the complete list of filter subscriptions from.
    extensions.adblockedge.whitelistschemes about chrome file irc keyword moz-safe-about news resource snews x-jsd addbook cid imap mailbox nntp pop data javascript List of URL schemes that shouldn't be blocked (applied to both whole documents and to item locations)
    extensions.adblockedge.disable_startup_delay true Enforce ABE being loaded completely at Firefox startup, even if delaying it
    Removed preferences
    extensions.adblockedge.blocklocalpages
    		Unused in Adblock edge
    extensions.adblockedge.blocktypes
    		Unused in Adblock edge
    extensions.adblockedge.checkedadblockinstalled
    		Unused in Adblock edge
    extensions.adblockedge.checkedadblockprefs
    		Unused in Adblock edge
    extensions.adblockedge.checkedadblocksync
    		Unused in Adblock edge
    extensions.adblockedge.checktoolbar
    		Unused in Adblock edge
    extensions.adblockedge.grouporder
    		Unused in Adblock edge
    extensions.adblockedge.linkcheck
    		Unused in Adblock edge
    extensions.adblockedge.linktypes
    		Unused in Adblock edger
    extensions.adblockedge.localschemes
    		Unused in Adblock edge
    extensions.adblockedge.noncollapsabletypes
    		Unused in Adblock edge
    extensions.adblockedge.patterns
    		Unused in Adblock edge
    extensions.adblockedge.protectchrome.*
    		Unused in Adblock edge
    extensions.adblockedge.settings_key
    		Unused in Adblock edge
    extensions.adblockedge.showintoolbar
    		Unused in Adblock edge
    extensions.adblockedge.showsubscriptions
    		Unused in Adblock edge
    extensions.adblockedge.synchronizationinterval
    		Unused in Adblock edge
    extensions.adblockedge.warnregexp
    		Unused in Adblock edge
    extensions.adblockEdge.subscriptions_fallbackerrors
    		Removed in Adblock edge 2.1.5
    extensions.adblockEdge.subscriptions_fallbackurl
    		Removed in Adblock edge 2.1.5
