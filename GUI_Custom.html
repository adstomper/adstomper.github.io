How can I put the Adblock Edge icon on my toolbar?

Usually this icon is added automatically the first time you install Adblock Edge, but maybe it was removed somehow afterwards. In Firefox you can customize the toolbar, simply right-click on the toolbar and choose "Customize". A window with icons will open, Adblock Edge icon should be one of them. Drag it with the mouse into the toolbar and place it where you want it to be. You can also reorder the toolbar icons by dragging them while the customization window is open.

How can I remove the Adblock Edge icon from my toolbar?

You don't like the icon or need the space for more important things? Click the arrow at the icon, then choose "Options" and "Show in tool bar" from the menu. There is also an Options menu in the Preferences dialog — just in case you decide to restore the icon. And by the way, you can also use the toolbar customization that is built into Firefox.



How do I change the icons displayed by Adblock Edge?

You can make your own style using this template:

@namespace url("http://www.mozilla.org/keymaster/gatekeeper/there.is.only.xul");

@-moz-document url("chrome://browser/content/browser.xul"),
      url("chrome://navigator/content/navigator.xul"),
      url("chrome://messenger/content/messenger.xul"),
      url("chrome://messenger/content/messageWindow.xul"),
      url("chrome://midbrowser/content/midbrowser.xul"),
      url("chrome://emusic/content/startup.xul"),
      url("chrome://webrunner/content/webrunner.xul"),
      url("chrome://global/content/customizeToolbar.xul") {

  toolbar[iconsize="small"] #abp-toolbarbutton,
  #PersonalToolbar #abp-toolbarbutton,
  #abp-status {
    list-style-image: url("file:///c:/icons/abe-16x16.png") !important;
  }

  #abp-toolbarbutton {
    list-style-image: url("file:///c:/icons/abe-24x24.png") !important;
  }
}

Here the images (one with three 16×16 icons and another with three 24×24 icons stacked vertically) are assumed to be located in the directory c:\icons, so you might need to change the addresses. To apply this style you can either use Stylish or edit the file chrome/userChrome.css in your profile directory (where is my profile directory?) and add this code. If userChrome.css doesn't exist yet you have to create it.

Note: The changes will take effect after you restart your browser.

How do I change the keyboard shortcut for ... / add a keyboard shortcut for disabling Adblock Edge?

There are three hidden preferences that can be accessed via about:config that define the keyboard shortcuts for Adblock Edge: extensions.adblockedge.sidebar_key (open/close blockable elements), extensions.adblockedge.settings_key (open Preferences dialog), extensions.adblockedge.enable_key (enable/disable Adblock Edge). You can change those to your liking.

Note: The changes will take effect after you restart your browser.
